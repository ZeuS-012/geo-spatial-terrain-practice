# TerraPrecision GIS Practice
An interactive 3D Geo-spatial terrain analysis tool built with React and Three.js.

## Features
- **3D Terrain Simulation**: Visualizes terrain mesh density and precision.
- **Service Comparison**: Compares NASA SRTM vs. LiDAR data models.
- **Flood Analysis**: Real-time water flow simulation on variable resolutions.

## Tech Stack
- React
- React Three Fiber (Three.js)
- Tailwind CSS
- Lucide Icons
